새 상품 등록

📋 상품 등록 가이드
① 기본 정보 → 상품명, 카테고리, 가격 입력 (필수)
② 노출 태그 → BEST, NEW IN 등 체크하면 해당 메뉴에 노출
③ 이미지 업로드 → "파일 업로드" 클릭 → 역할 선택(대표/감성/디테일/소재/무드) → 파일 선택. 올리면 위에 카드로 표시됨. ↑↓로 순서 변경
④ 컬러별 이미지 → 옵션에서 이름을 "컬러"로 입력 → 값에 색상명 입력 → 아래 컬러별 이미지 업로드 영역이 자동 생성
⑤ 에디토리얼 → 감성 카피 + 무드 타이틀. 상세페이지 중간 THE MOOD 섹션에 표시
⑥ 저장 → 저장 누르면 상세페이지에 바로 반영
기본 정보
영문, 숫자, 하이픈만. 상품명 입력 시 자동 생성

상세 페이지 블록

이미지와 텍스트를 순서대로 추가하면 상세 페이지에 쭉 나열됩니다.

추천 상품 ① 함께 보면 좋은 아이템

상세페이지 함께 보면 좋은 아이템 섹션에 노출돼요. 비워두면 같은 카테고리에서 자동 추천돼요.

추천 상품 ② Quietly Coveted (사랑받는 아이템)

상세페이지 Quietly Coveted 섹션에 노출돼요. 비워두면 다른 상품이 자동으로 떠요.

슬라이드 추가

이 영상 슬라이드가 보일 때 재생돼요. 비우면 음악 없음.
